# 📚 Library Management System

A full-stack Library Management System built using Java Spring Boot. This project helps manage books, students, book issues, returns, fines, payments, notifications, and library administration through a user-friendly interface.

## 🚀 Features

- 📖 Book Management (Add, Update, Delete)
- 👨‍🎓 Student Registration & Login
- 👨‍💼 Admin Dashboard
- 📚 Book Issue & Return
- 💳 Online Book Purchase
- 💰 Fine Management
- 📄 PDF & Image Upload
- 💬 Chat Support
- 🔔 Notifications
- 🔍 Search Books
- 📊 Dashboard Reports
- 🔐 Secure Authentication
- 📱 Responsive User Interface

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/Sandeep-Gautam83/library-management-systems.git
```

2. Open the project in IntelliJ IDEA.

3. Configure MySQL database in `application.properties`.

4. Run the project.

5. Open the browser:

```
http://localhost:8080
```
